# kAIbigan — MVP Design Walkthrough

## Welcome Page (Default Loader)
Animated logo + tagline: *"Your AI Kaibigan for Mental Wellness."* Sets emotional tone — warm, approachable, safe. Doubles as loader on slow networks so users never feel the app has frozen.

## Sign-In Page
Opens with a quick mood check-in *before* account creation. Delivers the core value proposition (emotional awareness) as the very first interaction. Reduces sign-up friction. Account setup or return login CTAs at the bottom.

## Home Page — Central Hub
Personalized, time-of-day greeting ("Good morning / evening"). Core features accessible in 1–2 taps: mood logs, AI chat, resource library, calendar. Low cognitive load — stressed students shouldn't hunt for anything.

### Mood Logs
Daily check-in — students pick a mood avatar and write a short note. Over time, users see emotional patterns; the avatar displayed on the home page reflects the most recent log. Functions like a mini diary that lives inside the app.

**Flow:** tap mood log button → modal opens → write note + pick avatar → submit → avatar becomes displayed home mood.

### Quick Actions
Mirrors the bottom nav bar on the home surface. Creates a "command center" feel — supports both in-content taps and nav-bar tappers. Reduces friction for different navigation preferences.

### Recent Chat History
Surfaces recent AI conversation threads on the home page so users can pick up where they left off. Mental health support often unfolds in continuing threads, not one-offs.

### Emergency Red Flag Protocol
Highly visible button at the bottom of the home page. Contrasting colors, clear iconography — in a moment of distress, users don't have to think. One tap surfaces crisis hotlines, emergency contacts, and de-escalation prompts. Designed with clinical input.

## Explore Page — Resource Library
Curated, localized library for the Filipino youth context. Built in partnership with local mental health advocates, universities, and schools. Category filters (academic stress, family pressure, relationships, self-care). Aim: the go-to library for Filipino students.

## Dedicated Chat Page
Heart of the app. Messaging-style UI familiar to Filipino students. AI responds with empathy-first language drawing from CBT and mindfulness frameworks — conversational, non-clinical. Safeguards: crisis / self-harm language triggers Red Flag resources.

## Calendar Page
Personal wellness planner. Color-coded moods per day. Reminders for self-care, self check-ins, and external events (exams, deadlines, family events). Builds proactive habits, not just crisis reactions. Pattern recognition: "I always feel down on Mondays."

## Account Page
Profile + preferences. Notification controls, privacy controls (export / delete data), emergency contact management. Summary of wellness journey — total mood logs, chat sessions, resources accessed — as positive reinforcement. Clean, friction-free.
