# kAIbigan Admin Dashboard Demo Checklist

## Run

- `npm run dev`
- `npm run build`

## 90-Second Path

- Open `/`
- Review KPI row and overview charts
- Open `/triage`
- Open `/crisis`
- Open `/wellbeing`

## Review Checklist

- Sidebar, topbar, and privacy badge appear on every route
- Date range selector changes overview KPI values
- Export button shows `PDF export generated`
- Crisis page shows only opaque case IDs and client-side filters work
- Wellbeing charts use the mobile mood colors
- No console errors during the walkthrough
